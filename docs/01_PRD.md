# Product Requirements Document (PRD)

## 1. Goal

Deliver a frictionless Word-of-the-Day experience that users can install, receive daily notifications from, and use long-term without needing to sign up — while preserving progress if they later choose to create an account.

---

## 2. Non-Goals (for v1)

- No social features
- No paid plans
- No admin UI (words are preloaded)
- No personalized word filtering yet
- No offline dictionary browsing beyond received words

---

## 3. Target Users

- Casual learners
- Language enthusiasts
- ESL users
- People who enjoy daily rituals

---

## 4. Core User Stories

- As a user, I want to receive one word per day at a time I choose
- As a user, I want to read past words I’ve already received
- As a user, I want to turn notifications off
- As a user, I want to install the app without creating an account
- As a user, I want to later create an account and keep my history

---

## 5. Success Metrics (Events Only)

- App installed
- Notification permission granted
- Daily notification delivered
- Word viewed
- Account created
- Notification disabled
